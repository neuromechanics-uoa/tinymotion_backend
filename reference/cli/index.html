    CLI Reference

    This page provides reference documentation for the command line interface.

    tinymotion-backend

    Command line interface to TinyMotion Backend

    Usage:

    tinymotion-backend [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --help boolean Show this message and exit. False

    Subcommands

    • consent: Commands to interact with consents in the backend.
    • infant: Commands to interact with infants in the backend.
    • user: Commands to interact with users in the backend.
    • version: Print the version of the TinyMotion backend and exit.
    • video: Commands to interact with videos in the backend.

    tinymotion-backend consent

    Commands to interact with consents in the backend.

    Usage:

    tinymotion-backend consent [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --help boolean Show this message and exit. False

    Subcommands

    • create: Create a new consent.
    • list: List existing consents.

    tinymotion-backend consent create

    Create a new consent.

    Usage:

    tinymotion-backend consent create [OPTIONS]

    Options:

    Name Type Description Default
    -u, --user-id uuid ID of the User to create the Consent _required
    -i, --infant-id uuid The infant_id of the infant this Consent relates to _required
    -n, --consent-giver-name text The Consent giver's name None
    -e, --consent-giver-email text The Consent giver's email None
    -p, --collected-physically boolean Flag to indicate if the Consent was collected physically False
    --help boolean Show this message and exit. False

    tinymotion-backend consent list

    List existing consents.

    Usage:

    tinymotion-backend consent list [OPTIONS]

    Options:

    Name Type Description Default
    -p, --pager boolean Display results using a pager False
    --help boolean Show this message and exit. False

    tinymotion-backend infant

    Commands to interact with infants in the backend.

    Options:

    Name Type Description Default
    --help boolean Show this message and exit. False

    Subcommands

    • create: Create a new infant.
    • delete: Delete the specified Infant.
    • list: List existing infants.
    • update: Update the specified Infant.

    tinymotion-backend infant create

    Create a new infant.

    Usage:

    tinymotion-backend infant create [OPTIONS]

    Options:

    Name Type Description Default
    -u, --user-id uuid ID of the User to create the Infant _required
    -f, --full-name text The infant's full name _required
    -n, --nhi-number text The infant's NHI number _required
    -b, --birth-date datetime (%Y-%m-%d) The infant's date of birth _required
    -d, --due-date datetime (%Y-%m-%d) The infant's expected date of birth _required
    --help boolean Show this message and exit. False
