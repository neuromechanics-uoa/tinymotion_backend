tinymotion-backend

Usage:

$ tinymotion-backend [OPTIONS] COMMAND [ARGS]...

Options:

  • --install-completion: Install completion for the current shell.
  • --show-completion: Show completion for the current shell, to copy it or customize the installation.
  • --help: Show this message and exit.

tinymotion-backend create-user

Create a new user.

Usage:

$ tinymotion-backend create-user [OPTIONS] EMAIL ACCESS_KEY

Arguments:

  • EMAIL: [required]
  • ACCESS_KEY: [required]

Options:

  • --disabled / --no-disabled: [default: no-disabled]
  • --help: Show this message and exit.
